# Chronacc F

A pat project. Time tracking system.

### How to use

1. Build

for front:

```sh
yarn install
```

for back:

```sh
yarn install
docker-compose up -d
yarn mig:run
```

2. Dev

for front:

```sh
yarn dev
```

for back:

```sh
yarn start:dev
```

### front stack highlights

react, next, tailwind, daisyui

### back stack highlights

nestjs, graphql, pg, typeorm, docker
